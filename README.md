## Run the Code

First, install the dependencies and run the development server:

```bash
npm run install # to install all dependencies

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
